# List Tools

Невеликий Python-проєкт для роботи зі списками.

## Функціональність

- Відбір парних чисел
- Прибирання дублікатів
- Обчислення середнього значення

## Встановлення

1. Клонуйте репозиторій:
2. Перейдіть у папку проєкту:
3. Запустіть скрипт:

```
git clone https://github.com/ТВІЙ-ЛОГІН/list-tools.git
```
```
cd list-tools
```
```
python lists.py
```
##Приклад використання

from lists import get_even_numbers, unique_sorted

data = [1, 2, 2, 3, 4, 5]

print(get_even_numbers(data))  # [2, 2, 4]

print(unique_sorted(data))     # [1, 2, 3, 4, 5]
