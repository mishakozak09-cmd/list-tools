# List Tools

Невеликий Python-проєкт для роботи зі списками.

## Функціональність

- Відбір парних чисел
- Прибирання дублікатів
- Обчислення середнього значення
- 
## Таблиця функцій

| Назва функції               | Опис                                   | Приклад виклику                          |
|:---------------------------:|:----------------------------------------|:-----------------------------------------|
| `get_even_numbers(numbers)` | Повертає список парних чисел            | `get_even_numbers([1, 2, 3, 4, 5, 6])`   |
| `get_unique_sorted(numbers)`| Повертає відсортований список без дублів| `get_unique_sorted([3, 1, 2, 2, 3])`     |
| `average(numbers)`          | Обчислює середнє значення списку        | `average([1, 2, 3, 4])`                  |

##Приклад використання

from lists import get_even_numbers, unique_sorted

data = [1, 2, 2, 3, 4, 5]

print(get_even_numbers(data))  # [2, 2, 4]

print(unique_sorted(data))     # [1, 2, 3, 4, 5]
